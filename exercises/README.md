## Structure

Each week's exercise folder contains:

- **Main exercise notebook** — problems with guidance and hints
- **`solutions/` sub-folder** — reference implementations to review after your attempt

```
exercises/
├── exercice1/
│   ├── exercise1.md
│   └── solutions/
│       └── exercise1.md
├── exercice2/
│   ├── exercise2.md
│   └── solutions/
│       └── exercise2.md
```
